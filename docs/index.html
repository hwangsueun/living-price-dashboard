생활물가 선행경보 대시보드

지역별 생활물가 부담을 실제 위험점수와 예측 위험점수로 나누어 확인하는 정책지원형 대시보드입니다. 지역 물가 흐름과 거시 금융환경을 종합해 선행압력지수를 계산하고, 분기별 위험 수준을 지도·순위·상세 화면으로 제공합니다.

위험점수 범위
0~100
분석 단위
지역 × 분기
주요 조정값
λ
표시 방식
지도 · 순위
지역별 위험 분포 예측-실제 비교 선행압력 조정 변수별 기여도

대시보드 화면 구성

🗺️ 지도 📊 순위 🔍 지역 상세 📘 지표 설명

지역 선택 예시

실제 점수 68.4
예측 점수 71.2
예측 차이 +2.8
선행압력 +6.1

실제 앱에서는 지도에서 지역을 클릭하거나 목록에서 지역을 선택하면 상세 지표가 갱신됩니다.

주요 기능

1. 지역별 위험 지도

전국 시도별 생활물가 위험점수를 0~100 기준 색상으로 표시합니다. 선택 지역은 경계선을 강조합니다.

2. 지역 순위 비교

실제 위험점수 또는 예측 위험점수 기준으로 지역별 순위를 비교합니다.

3. 지역 상세 분석

선택 지역의 실제 점수, 예측 점수, 예측 오차, 선행압력, 시계열 추이를 확인합니다.

4. 변수별 기여도

지역 물가 흐름과 거시 금융환경 변수가 예측 위험점수에 어떤 방향으로 영향을 줬는지 보여줍니다.

5. λ 조정

선행압력지수를 예측 위험점수에 얼마나 반영할지 조정합니다.

6. 경로 가중치 조정

지역 물가 흐름과 거시 금융환경의 반영 비중을 조정합니다.

지표 산출 구조

조정 선행압력지수 = 기본 선행압력지수 × λ
예측 위험점수 = 이전 분기 위험점수 + 조정 선행압력지수
위험점수는 값이 높을수록 생활물가 부담 위험이 큰 지역을 의미합니다. 0~40점은 낮음, 40~70점은 주의, 70~100점은 높음 수준으로 해석합니다.

로컬 실행 방법

git clone https://github.com/hwangsueun/living-price-dashboard.git
cd living-price-dashboard
pip install -r requirements.txt
streamlit run app.py
현재 구현은 Streamlit 기반입니다. GitHub Pages는 정적 HTML만 표시하므로 이 페이지는 프로젝트 소개용입니다. 실제 대시보드 앱을 웹에서 실행하려면 Streamlit Community Cloud, Render, Railway, Hugging Face Spaces 같은 실행형 배포 플랫폼을 사용해야 합니다.
